# Module 1: Responsive Web Design (HTML/CSS)
## 4-Week AI-Integrated Curriculum

**Philosophy:** Students don't need to memorize syntax — AI handles that. They need to:
1. **Read and understand** code AI generates
2. **Debug** when things break
3. **Prompt effectively** to get what they want
4. **Verify** the output is correct

**Structure:** 4 Wednesday sessions (1 hr) + 4 Saturday sessions (4 hrs)  
**Homework:** ~100 freeCodeCamp steps/week (~10 hrs)  
**Total freeCodeCamp:** ~400 steps (core concepts only)

---

## Assessment Strategy

### The Core Question
**Can the student read, debug, and verify code — even if AI wrote it?**

### Wednesday Checkpoints
1. **Read the code:** Given a snippet, explain what it does
2. **Predict the output:** Given HTML + CSS, sketch the result
3. **Find the bug:** Given broken code, identify the issue WITHOUT AI

### Saturday Verification
1. **Rebuild test:** Delete a component, rebuild without AI (45 min)
2. **Explain test:** Walk through code line-by-line to partner
3. **Prompt review:** Share what AI prompts worked/failed

### What We're NOT Testing
- Memorizing syntax
- Writing code from scratch without references
- Speed coding

## AI Prompting Guide for Students

### Good Debugging Prompts
- "Why isn't my flexbox centering vertically? Here's my code: [paste]"
- "This CSS selector isn't applying. What's wrong? [paste code]"
- "My form submit button doesn't do anything. What am I missing?"

### Good Building Prompts
- "Add a responsive navbar that collapses on mobile using Bootstrap"
- "Create a 3-column card grid using CSS Grid that stacks on mobile"
- "Style this form with proper spacing and hover states on the button"

### Bad Prompts (Too Vague)
- "Make it look better"
- "Fix my code"
- "Build me a website"

### Verification Questions to Ask AI
- "Explain what this CSS does line by line"
- "Why did you use flexbox instead of grid here?"
- "What happens to this layout at 600px screen width?"

## Bootstrap Quick Reference

### Grid Breakpoints
| Breakpoint | Class | Dimensions |
|------------|-------|------------|
| Extra small | (none) | <576px |
| Small | `sm` | ≥576px |
| Medium | `md` | ≥768px |
| Large | `lg` | ≥992px |
| Extra large | `xl` | ≥1200px |

### Common Patterns
