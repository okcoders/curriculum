Week 4 — Responsive Design & Bootstrap

This week focuses on making sites work on all screen sizes using Bootstrap — a CSS framework that gives you pre-built responsive components.


Why Bootstrap?

What it is: A CSS (and JS) library with pre-built classes for layout, components, and utilities.

Why use it:

  • Fast prototyping — build UIs in minutes
  • Responsive by default — works on mobile out of the box
  • Consistent styling — buttons, forms, cards all match
  • Huge ecosystem — tons of templates, examples, docs
  • AI knows it well — generated code often uses Bootstrap

Trade-offs:

  • Adds ~25-50kb to your page
  • Sites can look "Bootstrap-y" without customization
  • Less control than writing CSS from scratch


Adding Bootstrap to Your Project

Option 1: CDN (easiest)

<!-- In your <head> -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Before closing </body> (for JS components) -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"></script>

Use when: Quick projects, learning, prototypes.

Option 2: npm install

npm install bootstrap

Use when: Production apps, need to customize, using a build tool.

Responsive Breakpoints

Bootstrap has 6 breakpoints. Add the breakpoint to column classes to change layout at different screen sizes.

Breakpoint Class infix Screen width
Extra small(none)<576px
Smallsm≥576px
Mediummd≥768px
Largelg≥992px
Extra largexl≥1200px
XXLxxl≥1400px

Responsive Column Classes

<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        Card
    </div>
</div>

/* Translation:
   - Mobile (default): full width (12 columns)
   - Tablet (md+): half width (6 columns)  
   - Desktop (lg+): third width (4 columns)
*/

Resize your browser to see this change:

col-12 col-md-12 col-lg-4

Question: What does col-sm-6 col-lg-3 mean?

Answer: Full width on extra-small (default), half width on small screens and up, quarter width on large screens and up.



Container Types

.container       /* Fixed max-width at each breakpoint */
.container-fluid /* Full width always */
.container-md    /* Full width until md, then fixed */

When to use:

  • .container — most sites, content stays centered with margins
  • .container-fluid — dashboards, full-bleed designs

Bootstrap Utility Classes

Quick styling without writing CSS. These are your most-used utilities.

Spacing (Margin & Padding)

/* Pattern: {property}{sides}-{size} */

/* Properties */
m = margin
p = padding

/* Sides */
t = top
b = bottom
s = start (left)
e = end (right)
x = left & right
y = top & bottom
(blank) = all sides

/* Sizes: 0, 1, 2, 3, 4, 5, auto */

/* Examples */
.mt-3     /* margin-top: 1rem */
.px-4     /* padding-left & padding-right: 1.5rem */
.mb-0     /* margin-bottom: 0 */
.p-5      /* padding: 3rem on all sides */
.mx-auto  /* center horizontally */

Display

.d-none           /* display: none */
.d-block          /* display: block */
.d-flex           /* display: flex */
.d-inline         /* display: inline */

/* Responsive */
.d-none .d-md-block   /* hidden on mobile, visible on tablet+ */
.d-md-none            /* visible on mobile, hidden on tablet+ */

Flexbox Utilities

.d-flex                   /* enable flexbox */
.justify-content-center   /* center horizontally */
.justify-content-between  /* space between */
.align-items-center       /* center vertically */
.flex-column              /* stack vertically */
.gap-3                    /* gap between items */

Text

.text-center    /* center text */
.text-start     /* left align */
.text-end       /* right align */
.fw-bold        /* font-weight: bold */
.fs-1 to .fs-6  /* font sizes (1 = largest) */
.text-muted     /* gray text */
.text-primary   /* blue text */
.text-danger    /* red text */

Background & Borders

.bg-primary, .bg-secondary, .bg-success, .bg-danger
.bg-light, .bg-dark, .bg-white

.border
.border-primary
.rounded        /* border-radius */
.rounded-pill   /* fully rounded */
.shadow         /* box shadow */
.shadow-lg      /* larger shadow */

Question: How would you center a div horizontally with Bootstrap?

Answer: mx-auto (sets margin-left and margin-right to auto). The element needs a set width or d-block.



Responsive Utilities

Most utilities can be made responsive by adding a breakpoint.

/* Show/hide at breakpoints */
.d-none .d-md-block     /* hidden mobile, show tablet+ */
.d-block .d-lg-none     /* show until large, then hide */

/* Text alignment */
.text-center .text-md-start  /* center on mobile, left on tablet+ */

/* Flex direction */
.flex-column .flex-md-row    /* stack on mobile, row on tablet+ */

/* Spacing */
.p-2 .p-md-4                 /* less padding mobile, more on tablet+ */

Common pattern — mobile menu:

<!-- Desktop nav -->
<nav class="d-none d-lg-flex">...</nav>

<!-- Mobile hamburger -->
<button class="d-lg-none">☰</button>


Customizing Bootstrap

Override with Your Own CSS

<!-- Load Bootstrap first -->
<link href="bootstrap.min.css" rel="stylesheet">
<!-- Then your overrides -->
<link href="custom.css" rel="stylesheet">
/* custom.css */
.btn-primary {
    background-color: #7c3aed;  /* your brand purple */
    border-color: #7c3aed;
}

.navbar {
    padding: 1rem 2rem;  /* more padding */
}

Use Bootstrap's Variables (Advanced)

/* If using Sass */
$primary: #7c3aed;
$font-family-base: 'Inter', sans-serif;

@import "bootstrap/scss/bootstrap";


Reading Bootstrap Docs

Bootstrap's documentation is excellent. Get comfortable navigating it.

Key sections:

  • Layout → Grid — column system
  • Components — buttons, cards, navbars, modals
  • Utilities — spacing, display, flex, text
  • Examples — full page templates to learn from

Pro tip: When AI generates Bootstrap code you don't recognize, search the docs. Understanding the class names helps you customize and debug.



Bootstrap vs Writing CSS

Same result, different approaches:

Custom CSS:

.card-grid {
    display: flex;
    flex-wrap: wrap;
    gap: 1rem;
}
.card-grid > div {
    flex: 1 1 300px;
}

Bootstrap:

<div class="row g-3">
    <div class="col-12 col-md-6 col-lg-4">...</div>
    <div class="col-12 col-md-6 col-lg-4">...</div>
    <div class="col-12 col-md-6 col-lg-4">...</div>
</div>

When to use which:

  • Bootstrap: rapid prototyping, admin dashboards, standard layouts
  • Custom CSS: unique designs, performance-critical, learning fundamentals
  • Both: Bootstrap for structure, custom CSS for brand-specific styling
