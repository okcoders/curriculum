Module 3 Week 3 — Supabase Database

This week we replace our in-memory array with a real PostgreSQL database using Supabase.

What is Supabase?

Supabase is a hosted PostgreSQL database with:

  • Dashboard UI for managing tables
  • JavaScript client library
  • Auto-generated REST API
  • Authentication (optional)
  • Free tier for development

Setup

  1. Create account at supabase.com
  2. Create a new project (pick a password, choose nearest region)
  3. Go to Settings → API and find:
    • Project URL
    • anon public key

Create Table

In Supabase dashboard, use the Table Editor to create an expenses table:

Column Type Notes
id int8 Primary key, auto-increment
description text
amount numeric
category text
date date
created_at timestamptz Default: now()

Install Client

npm install @supabase/supabase-js dotenv

Connect in Express

require("dotenv").config();
const { createClient } = require("@supabase/supabase-js");

const supabase = createClient(
  process.env.SUPABASE_URL,
  process.env.SUPABASE_KEY
);

Environment Variables

Create a .env file in your project root:

SUPABASE_URL=https://xxxxx.supabase.co
SUPABASE_KEY=your-anon-key-here
Important:
  • Add .env to .gitignore — never commit secrets!
  • require("dotenv").config() must be at the TOP of your file

CRUD Operations

Select All

app.get("/api/expenses", async (req, res) => {
  const { data, error } = await supabase
    .from("expenses")
    .select("*");
  
  if (error) {
    return res.status(500).json({ error: error.message });
  }
  res.json(data);
});

Select One

app.get("/api/expenses/:id", async (req, res) => {
  const { data, error } = await supabase
    .from("expenses")
    .select("*")
    .eq("id", req.params.id)
    .single();
  
  if (error) {
    return res.status(404).json({ error: "Not found" });
  }
  res.json(data);
});

Insert

app.post("/api/expenses", async (req, res) => {
  const { description, amount, category } = req.body;
  
  const { data, error } = await supabase
    .from("expenses")
    .insert({ description, amount, category })
    .select()
    .single();
  
  if (error) {
    return res.status(500).json({ error: error.message });
  }
  res.status(201).json(data);
});
Note: Add .select().single() after .insert() to get the created row back!

Update

app.put("/api/expenses/:id", async (req, res) => {
  const { description, amount, category } = req.body;
  
  const { data, error } = await supabase
    .from("expenses")
    .update({ description, amount, category })
    .eq("id", req.params.id)
    .select()
    .single();
  
  if (error) {
    return res.status(500).json({ error: error.message });
  }
  res.json(data);
});

Delete

app.delete("/api/expenses/:id", async (req, res) => {
  const { error } = await supabase
    .from("expenses")
    .delete()
    .eq("id", req.params.id);
  
  if (error) {
    return res.status(500).json({ error: error.message });
  }
  res.status(204).send();
});

Query Filters

// Filter by category
.from("expenses").select("*").eq("category", "Food")

// Greater than
.from("expenses").select("*").gt("amount", 100)

// Order by
.from("expenses").select("*").order("created_at", { ascending: false })

// Limit results
.from("expenses").select("*").limit(10)

Common Mistakes

Mistake Problem Fix
data is null Query failed Check the error object
Column not found Typo in column name Check table schema matches code
Env var undefined Missing dotenv require("dotenv").config() at top
Insert returns null Missing .select() Add .select().single()
