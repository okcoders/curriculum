Bootstrap

Bootstrap is a popular open-source front-end framework that helps developers create responsive, mobile-first websites quickly. It includes pre-designed components, CSS styles, and JavaScript functionality.

  • CSS Framework
  • Mobile-first
  • Responsive Design
  • Pre-designed components
  • Easy to use




Setup

To get started with Bootstrap, you need to include the Bootstrap CSS and JavaScript files in your HTML document. You can either download the files and host them on your own server, or include them from a CDN (Content Delivery Network).

CDN

To include Bootstrap from a CDN, you can use the following HTML code:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/css/bootstrap.min.css">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.2.0/dist/js/bootstrap.bundle.min.js"></script>

Breakpoints

Bootstrap uses a responsive grid system that allows you to specify different column widths at different screen sizes. This is done using breakpoints, which are defined in the Bootstrap CSS file.


  • xs - Extra small devices (phones)
  • sm - Small devices (tablets)
  • md - Medium devices (laptops)
  • lg - Large devices (desktops)
