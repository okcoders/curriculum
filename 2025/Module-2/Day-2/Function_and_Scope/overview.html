Functions

Function - A block of reusable code designed to perform a specific task. A function can take inputs (parameters), process them, and return an output.

Function Declaration - A way to define a function using the function keyword. Function declarations are hoisted to the top of their scope, meaning they can be called before they are defined in the code.

Function Expression - A function assigned to a variable. Function expressions are not hoisted, so they must be defined before they are called.

Arrow Function - A shorter way to write functions introduced in ES6. Arrow functions do not bind their own this value, so they inherit this from the surrounding scope.

Parameters - Variables listed in the function definition.

Arguments - Values passed into the function when it's called.

Return Statement - The return statement ends the function execution and sends a value back to the caller.

Scope

Scope - The context in which variables and functions are accessible. Scope determines where in the code a variable or function can be used.

Global Scope - Variables declared outside any function or block are in the global scope and can be accessed from anywhere in the code.

Function Scope - Variables declared inside a function are only accessible within that function.

Block Scope - Variables declared with let and const inside a block {} are only available within that block.

Hoisting - JavaScript moves function declarations and var declarations to the top of their scope during the compilation phase.

Return Statement - The return statement ends the function execution and sends a value back to the caller.
