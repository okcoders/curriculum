Booleans, Logical Operators, Objects, and Switch Statements

A Boolean is a data type that can have one of two values: true or false. Booleans are used in logic and conditional statements to make decisions in programs.

Logical Operators are used to combine two or more conditions.

  • && (AND): Both conditions must be true for the whole expression to be true.
  • || (OR): At least one condition must be true for the whole expression to be true.
  • ! (NOT): Reverses the boolean value.

Objects are collections of key-value pairs where the key is a string (or symbol) and the value can be any data type. Objects allow us to store and organize related data together.

A Switch Statement evaluates an expression and compares it with several possible cases. It is often used when you need to compare a single variable against different values.
