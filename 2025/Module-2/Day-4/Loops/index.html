For loop - Use a for loop when you know exactly how many times you want to repeat an action — like counting, iterating over a list, or running timed tasks.

While loop - Use a while loop when you don’t know how many times you’ll loop — just that you want to continue as long as a condition is true.

Do While loop - Use do...while if you want the code to run at least once, no matter what.

Nesting loops - Use nested loops when you need to repeat actions inside other actions — like comparing every pair in a list, or working with rows and columns.

Break - used to stop a loop early.

Continue - used to skip an iteration without stopping the whole loop.

Return - immediately exits the entire function, not just the loop. It ends the function’s execution and optionally sends back a value.

Ternary Operator - Use the ternary operator as a shortcut for simple if/else decisions.

Built-in functions - help you manipulate data easily without writing everything from scratch.

• String functions (.toUpperCase(), .includes()) • Array functions (.push(), .map(), .filter()) • Math functions (Math.random(), Math.floor())

Recursion - a technique where a function calls itself to solve a smaller version of a problem. It’s powerful when working with data that has repeated or nested structures — like navigating folders, parsing HTML, or solving puzzles.
